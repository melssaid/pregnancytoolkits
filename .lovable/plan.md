# خطة تحسين شريط الشراء المميز + صندوق التوصيات الذكي

## المشكلة الحالية

1. **FooterCard** (شريط "الوصول المميز"): النص صغير جداً (10px)، لا يوجد زر شراء واضح، كلمة "عرض الخطط" مخفية في chip صغير، والوصف غير bold
2. **صندوق البداية**: عند اختيار المرحلة في الـ Onboarding لا يفتح صندوق توصيات مخصص بالأدوات المناسبة

---

## الجزء الأول: إعادة تصميم شريط الشراء (FooterCard)

### التغييرات في `src/pages/Index.tsx`:

**التصميم الجديد — مستوحى من Flo, Clue, Headspace:**

```text
┌──────────────────────────────────┐
│ 👑  احصلي على Premium            │
│                                  │
│  ✓ 60 تحليل ذكي شهرياً          │
│  ✓ جميع الأدوات مفتوحة (36+)    │
│  ✓ نصائح مخصصة لحملك            │
│                                  │
│  ┌──────────────────────────┐    │
│  │   اشتركي الآن — $2.99/شهر │    │
│  └──────────────────────────┘    │
│  تجربة مجانية · إلغاء أي وقت    │
└──────────────────────────────────┘
```

- أيقونة Crown أكبر (w-12 h-12) مع gradient ذهبي
- عنوان bold بخط 14px: "احصلي على Premium" بدل "الوصول المميز"
- 3 نقاط مميزات مع ✓ بخط واضح (12px, font-semibold)
- **زر شراء كبير وواضح** (h-11, rounded-xl, gradient) بنص "اشتركي الآن"
- سطر ثقة أسفل الزر: "إلغاء أي وقت · تجربة مجانية"
- عند نفاد التحليلات: شريط أحمر مع نص "نفدت تحليلاتك — اشتركي الآن!"
- قم بتقليد افضل نظام عالمي للإشتراكات بألوان مناسبة وجذابة

## الجزء الثاني: صندوق توصيات ذكي بعد اختيار المرحلة

### ملف جديد: `src/components/onboarding/JourneyRecommendations.tsx`

عند اختيار المرحلة في الخطوة 2 من الـ Onboarding، يظهر صندوق مقترحات أسفل الاختيار:

**إذا اختارت "حامل":**

```text
┌─ 🤰 أدوات مهمة لحملك ──────────┐
│  📅 حاسبة موعد الولادة           │
│  🏃 تمارين آمنة للحامل           │
│  🍎 دليل التغذية والفيتامينات    │
│  👶 متابعة نمو الجنين أسبوعياً 
وأضف لها بقية الأدوات المرتبطة بدون أيقونات مزعجه│
└───────────────────────────┘
```

**إذا اختارت "تخطط للحمل" (Fertility):**

```text
┌─ 🌸 أدوات تساعدك على الحمل ─────┐
│  📊 متابعة الدورة والتبويض       │
│  ✅ فحوصات ما قبل الحمل          │
│  📚 أكاديمية الخصوبة             │
└──────────────────────────────────┘
```

**إذا اختارت "أم جديدة" (Postpartum):**

```text
┌─ 👶 أدوات لك ولطفلك ────────────┐
│  🍼 متابعة الرضاعة               │
│  😴 متابعة نوم الطفل             │
│  💜 دعم الصحة النفسية بعد الولادة│
│  🧷 متابعة حفاضات الطفل          │
└──────────────────────────────────┘
```

### تعديل `src/components/onboarding/OnboardingStep2Journey.tsx`:

- إضافة قسم المقترحات أسفل اختيار المرحلة مباشرة
- عرض 3-4 أدوات مقترحة كـ chips بألوان ناعمة
- بدون تفاعل — فقط إعلامي لتعريف المستخدمة بالأدوات المتاحة
- اجعل الخطوط واضحة ومتباينة مع الخلفية في كل الصندوق
