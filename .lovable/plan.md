

# خطة إنشاء 9 صور إنستجرام كاروسيل تعليمي لتطبيق Pregnancy Toolkits

## فلسفة التصميم: "Tender Bloom"

فلسفة بصرية تجمع بين الدفء الأمومي والأناقة الطبية، مستوحاة من ألوان التطبيق الحالية (الوردي الغني #D4657E، الأخضر المريمي، البنفسجي الناعم).

## الصور التسع (1080×1080 بكسل)

| # | العنوان | المحتوى |
|---|---------|---------|
| 1 | **غلاف الكاروسيل** | شعار التطبيق + "42+ أداة ذكية لرحلة الأمومة" + عناصر بصرية مجردة |
| 2 | **المساعد الذكي** | AI Assistant — أيقونة دماغ/رسائل + "اسألي أي سؤال عن حملك" |
| 3 | **حاسبة موعد الولادة** | تقويم + عد تنازلي بصري أنيق |
| 4 | **عداد ركلات الجنين** | قدم طفل + رسم بياني بسيط للحركة |
| 5 | **تتبع الدورة والخصوبة** | حلقة دورة ملونة + نافذة الخصوبة |
| 6 | **التغذية والوجبات** | أيقونات طعام + "خطط وجبات مخصصة لكل ثلث" |
| 7 | **خطة الولادة** | مستند أنيق + "خطة ولادة شاملة بالذكاء الاصطناعي" |
| 8 | **الخصوصية والأمان** | درع + "بياناتك على جهازك فقط" |
| 9 | **دعوة للتحميل (CTA)** | "ابدئي الآن — مجاني 100%" + رابط التطبيق |

## التنفيذ التقني

1. **إنشاء ملف فلسفة التصميم** → `/mnt/documents/tender-bloom-philosophy.md`
2. **إنشاء 9 صور PNG** (1080×1080) باستخدام Python + Pillow مع:
   - خطوط: Lora (عناوين) + InstrumentSans (نصوص) من الخطوط المتاحة
   - لوحة ألوان التطبيق: Rose #D4657E، Sage #6DB88F، Lavender #E8D5E8، أبيض
   - أشكال هندسية ناعمة (دوائر، أقواس، تدرجات)
   - نص بسيط جدًا — التصميم البصري هو البطل
3. **فحص الجودة** → تحويل لصور ومراجعة بصرية
4. **التسليم** → 9 ملفات PNG + ملف الفلسفة في `/mnt/documents/`
